# Setup

### Frontend
```
cd mms-frontend
```
Install dependencies
```
npm i
```
Run server
```
npm run dev
```

### Backend
```
cd mms-backend
```
Create a virtual environment
```
python -m venv venv
```
Activate virtual environment
```
source venv/bin/activate
```
Install dependencies
```
pip install -r requirements.txt
```
Run migrations
```
python manage.py mnigrate
```
Run server
```
python manage.py runserver
```
